01 — Resumen Ejecutivo

Qué es Advance Estate CRM

CRM inmobiliario con IA diseñado para RE/MAX Advance República Dominicana. Arquitectura multi-tenant orientada a ser vendible a otras franquicias y agentes.

Propuesta de valor

Advance Estate CRM es un sistema de gestión de relaciones con clientes específicamente diseñado para el mercado inmobiliario de República Dominicana. Integra Ava, un agente de IA que opera en WhatsApp de forma autónoma para calificar leads entrantes, y un CRM web para que los agentes humanos gestionen todo el ciclo de venta. El sistema está construido como producto SaaS multi-tenant con el objetivo de ser vendible a otras franquicias RE/MAX y agencias independientes.

🎯 Mercado objetivo

Franquicias RE/MAX y agencias inmobiliarias de República Dominicana y Caribe hispanoparlante. Ticket promedio de propiedades: USD 150K–2M+.

🚀 Estado actual

En producción en Vercel (frontend) + Railway (Ava IA). CI/CD automático desde GitHub. Base de datos en Supabase PostgreSQL con 4 migraciones aplicadas.

Seguridad — Capas implementadas

  • 🔐 middleware.ts (edge) — Supabase Auth en todas las rutas /dashboard/*
  • 🔑 AVA_WEBHOOK_SECRET — Shared secret Railway↔Vercel. Bearer token en cada llamada POST /api/ava
  • 🛡️ UUID validation — contact_id validado con regex antes de cualquier query a Supabase
  • 🧹 sanitizeField() — Limpia caracteres de control, trunca a límite seguro para prevenir prompt injection
  • 📦 RLS en Supabase — Row Level Security activo en todas las tablas principales
